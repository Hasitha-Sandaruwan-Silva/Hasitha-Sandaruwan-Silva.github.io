01 — ABOUT

Who am I

I'm a full-stack developer from Sri Lanka who builds complete, production-ready systems. I work across the full spectrum — REST APIs, frontend UIs, desktop applications, and databases.

My most complex project is Mahameruwa Leisure Hotel Management System — a multi-role platform built with Django REST Framework and Next.js 14, handling everything from OTP authentication to real-time inventory and finance reporting.

I follow clean architecture patterns (MVC, REST), use tools like Postman and MySQL Workbench for testing, and write code that actually works in the real world.
